# 🚀 API de Usuários - Spring Boot

API REST desenvolvida com Java + Spring Boot, com operações completas de CRUD e integração com PostgreSQL.

## 🧰 Tecnologias
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## 📌 Funcionalidades
- Criar usuário
- Listar usuários
- Buscar por ID
- Atualizar usuário
- Deletar usuário

## 🔗 Endpoints

| Método | Rota | Descrição |
|--------|------|----------|
| GET | /usuarios | Listar todos |
| GET | /usuarios/{id} | Buscar por ID |
| POST | /usuarios | Criar |
| PUT | /usuarios/{id} | Atualizar |
| DELETE | /usuarios/{id} | Deletar |

## ▶️ Como rodar

1. Clonar repositório
2. Configurar banco PostgreSQL
3. Rodar aplicação
